# Required Libraries

Download these files manually or use CDN links:

1. **TweetNaCl.js** - https://cdn.jsdelivr.net/npm/tweetnacl@1.0.3/nacl-fast.min.js
2. **qrcode.js** - https://cdn.jsdelivr.net/npm/qrcodejs@1.0.0/qrcode.min.js
3. **JSZip** - https://cdn.jsdelivr.net/npm/jszip@3.10.1/dist/jszip.min.js
4. **FileSaver.js** - https://cdn.jsdelivr.net/npm/file-saver@2.0.5/dist/FileSaver.min.js

For offline use, download these files and place them in this directory.
